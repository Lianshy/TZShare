## 谈资微信分享封装

## Example

```javascript
<scirpt src="@/lib/tz-share.min.js"></scirpt>
<script>
    new TZShare({options})
</script>
```

### options 
{ key:default }

> link: "分享链接"

> imgUrl: "分享图标"

> title: '分享标题'

> desc: "分享描述"

> [jsApiList](https://developers.weixin.qq.com/doc/offiaccount/OA_Web_Apps/JS-SDK.html) : ['onMenuShareAppMessage', 'onMenuShareTimeline', 'onMenuShareQQ', 'onMenuShareQZone']
